Movies example for om-autocomplete

  • Results are fetched from omdbapi.com. The images are randomly fetched from lorempixel.com because I could not find any free API to get movie posters.
  • There is no hovering or selecting in this demo -- if the input loses focus, the suggestions are still displayed, and clicking one of them just opens a page with details.
  • Open DevTools to watch requests get canceled if you type fast enough.
